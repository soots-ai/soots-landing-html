# Template Pages

This folder contains all the unused HTML template files from the original Soots template. These files are kept for future reference but are not currently being used in the Soots project.

## Currently Used Pages (in root directory)

The following 9 pages are actively used in the Soots project:

- `index.html` - Main homepage
- `lead-generation.html` - Lead generation page
- `contact-us-page.html` - Contact page
- `signup.html` - Signup page
- `faq-page.html` - FAQ page
- `terms-conditions-page.html` - Terms & conditions
- `privacy-page.html` - Privacy policy
- `refund-policy-page.html` - Refund policy
- `gdpr-page.html` - GDPR compliance

## Usage

If you need to add new pages to the Soots project in the future, you can:

1. Copy the relevant template file from this folder
2. Rename it appropriately
3. Update the content for Soots branding
4. Add it to the Vite configuration in `vite.config.js`
5. Update navigation links as needed

## Note

These template files maintain their original Soots branding and content. They would need to be updated with Soots branding and content before being used in the project.
